# InspireSurety Color System

## Hierarchy Logic
- Level 1 (Hub) = darkest/most saturated = category authority
- Level 2 (Guide) = same hue, brighter/lighter = educational zone
- Level 3 (Bond Page) = same hue, distinct accent shift = transactional zone

All levels share the same base hue family so users feel connected navigation, but gradient direction and saturation shift signals which "zone" they're in.

## Class Naming Convention
- Hub pages: `.is-[category-abbr]` e.g. `.is-lp` (License & Permit)
- Guide pages: `.is-[category-abbr]g` e.g. `.is-lpg`
- Bond pages: `.is-[category-abbr]b` e.g. `.is-lpb`

## Shared Structure (all levels)
- Full-bleed hero with gradient background
- Floating dark TOC card overlapping hero
- Alternating white/gray/tint sections
- Numbered reason/step cards
- FAQ accordion
- Full-bleed final CTA matching level gradient
